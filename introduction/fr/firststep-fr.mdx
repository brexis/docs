## Comment démarrer avec FedaPay

Bienvenue sur **FedaPay**, la solution tout-en-un pour accepter des paiements en ligne en toute simplicité. Que vous soyez une entreprise cherchant à intégrer des paiements sur votre site ou un particulier souhaitant recevoir de l'argent rapidement, FedaPay vous accompagne à chaque étape du processus.

Pour commencer à utiliser FedaPay, suivez ce guide rapide en 3 étapes ! Que vous choisissiez d’envoyer des liens de paiement à vos clients ou d’intégrer notre système directement à votre plateforme, FedaPay rend l’expérience fluide et facile.

### Scénario 1: Recevoir des paiements rapidement avec un lien de paiement

Si vous avez besoin de recevoir de l’argent de vos clients sans intégration technique complexe, **FedaPay** est votre solution. Il vous permet de générer des liens de paiement sécurisés en quelques minutes, que vous pouvez envoyer directement à vos clients.

**3 étapes simples** :

1. **Créer un client**
    - Depuis votre tableau de bord FedaPay, renseignez les informations de base du client : **Nom, Prénom, Email** (l'email est unique et essentiel pour suivre vos clients).
    - Le numéro de téléphone est facultatif.

2. **Créer une transaction**
    - Une transaction représente le paiement que vous attendez du client. Il vous suffit d’indiquer le **montant**, **une description** (obligatoire) et, si vous le souhaitez, un lien de retour personnalisé.

3. **Générer un lien de paiement**
    - Après avoir créé la transaction, un lien de paiement sécurisé sera généré. Ce lien peut être envoyé au client par email ou tout autre moyen de communication (WhatsApp, SMS, etc.).
    - Ce lien est **valable pendant 24 heures** et garantit une transaction sécurisée et validée par FedaPay.

### Scénario 2: Intégrer FedaPay à votre plateforme e-commerce

Pour les entreprises qui souhaitent accepter des paiements directement sur leur site web ou application mobile, **FedaPay** offre une solution clé en main. Vous pouvez rapidement intégrer un formulaire de paiement sécurisé avec notre module **Feda Checkout**, qui combine HTML, JavaScript et CSS.

**3 étapes pour intégrer FedaCommerce :**

1. **Tester Feda Checkout**
    - Faites un test rapide pour voir comment le processus fonctionne sur votre site. Remplissez un formulaire de paiement de test avec des informations comme un **nom**, **prénom** et **email** valide.

2. **Ajout d’un bouton de test ici pour une meilleure expérience utilisateur.**

3. **Intégrer le code de paiement**
    - Ajoutez le code d'intégration à votre page de paiement. Ce code est conçu pour être simple à intégrer et sécurisé.
    - Assurez-vous de remplacer **VOTRE_CLE_API_PUBLIQUE** par votre clé API publique que vous trouverez dans votre compte FedaPay.

    _**Conseil** : Utilisez toujours la clé API publique pour les intégrations côté client et gardez la clé API privée pour votre serveur._

4. **Connecter votre compte à Feda Checkout**
    - Une fois le code intégré, vous devrez connecter votre compte FedaPay à Feda Checkout pour que votre site puisse traiter les paiements.
    - Cette étape est simple et ne doit être effectuée qu'une seule fois. Une fois votre compte lié, vos utilisateurs pourront effectuer des paiements en un clic depuis votre site ou application.
