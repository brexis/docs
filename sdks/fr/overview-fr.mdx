---
title: 'Aperçu des SDKs FedaPay'
---

Découvrez les SDKs de FedaPay conçus pour des intégrations de paiements fluides. Avec plusieurs SDKs côté serveur et front-end, les développeurs peuvent gérer facilement les transactions et personnaliser les fonctionnalités de paiement selon leurs besoins.

## SDKs côté serveur

Les SDKs côté serveur permettent aux développeurs backend d'intégrer efficacement les fonctionnalités de FedaPay à leurs applications. Ces bibliothèques offrent des outils pour gérer les clients, effectuer des transactions et interagir avec l'API FedaPay.

<CardGroup cols={2}>
  <Card title="PHP" icon="php" href="/sdks/fr/php-fr">
    Une bibliothèque complète pour les développeurs PHP, compatible avec des frameworks tels que Laravel et Symfony.
  </Card>
  <Card title="Node.js" icon="node" href="/sdks/fr/nodejs-fr">
    Conçu pour les environnements JavaScript côté serveur, ce SDK simplifie les interactions avec l'API dans les applications Node.js.
  </Card>
  <Card title="Ruby" icon="gem" href="/sdks/fr/ruby-fr">
    Adapté aux développeurs Ruby et Ruby on Rails, ce SDK facilite l'intégration avec l'API et la gestion des paiements.
  </Card>
</CardGroup>

## SDKs Web 

Les SDKs front-end permettent aux développeurs d'intégrer aisément des options de paiement à leurs applications web tout en tirant parti des widgets et interfaces préconçus.

<CardGroup cols={2}>
  <Card title="React" icon="angular" href="/sdks/nodejs">
    Une bibliothèque React pour intégrer le bouton et le widget de paiement FedaPay, idéale pour les applications web modernes.
  </Card>
  <Card title="Angular" icon="react" href="/sdks/python">
    Conçu pour les projets Angular, ce SDK garantit une intégration fluide des fonctionnalités de paiement FedaPay grâce à des composants adaptés. 
  </Card>
</CardGroup>

<Note>
Cliquez sur un SDK pour consulter sa documentation, ses instructions d'installation et ses cas d'utilisation adaptés à vos besoins de développement.
</Note>
