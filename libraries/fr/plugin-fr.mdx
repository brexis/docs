## WooCommerce

Intégrer FedaPay à votre boutique WooCommerce est simple grâce à notre plugin dédié pour WordPress. Suivez ce guide pour installer et configurer le plugin afin de commencer à accepter des paiements en ligne rapidement.

<Steps>
  <Step title="Télécharger et installer le plugin">
    Vous pouvez télécharger le plugin FedaPay :

    - Directement depuis ce [Lien](https://wordpress.org/plugins/woo-gateway-fedapay/)
    - Ou depuis votre tableau de bord WordPress, en allant dans **Extensions** > **Ajouter** et en recherchant **"FedaPay WooCommerce"**.

    _**![(Image montrant le plugin dans WordPress)]**_

  </Step>
  <Step title="Activer le plugin">
    Une fois le plugin installé, activez-le en cliquant sur **Activer**. Ensuite, dirigez-vous vers les paramètres de WooCommerce pour finaliser la configuration.

    _**![(Image montrant l'activation du plugin)]**_
  </Step>
  <Step title="Accéder aux paramètres de WooCommerce">
    Pour configurer FedaPay, allez dans le tableau de bord WordPress puis cliquez sur :

    - **WooCommerce**
    - **Réglages**
    - **Paiements**

  Dans cette section, vous verrez tous les moyens de paiement disponibles. Tout en bas, vous trouverez **FedaPay**.

  _**![(Image montrant l'onglet Paiements dans WooCommerce)]**_

  </Step>
  <Step title="Configurer la      passerelle FedaPay">
    Cliquez sur **Configuration** à côté de FedaPay pour ouvrir les options de configuration.

    Voici les principaux paramètres à configurer :

    - **Activer FedaPay**: Cochez cette option pour activer FedaPay et recevoir des paiements en direct.
    - **Mode Test (sandbox)** : Cochez cette case uniquement si vous effectuez des tests. Laissez-la décochée lorsque vous passez en mode live pour accepter les paiements réels.


    _**![(Image montrant l'interface de configuration de FedaPay)]**]
  </Step>
  <Step title="Récupérer et entrer vos clés API">
    Pour utiliser FedaPay, vous devez entrer vos clés API (privées) pour les environnements test et live. Voici comment les récupérer :

    1. Connectez-vous à votre compte FedaPay.
    2. Accédez à votre **tableau de bord** FedaPay.
    3. Trouvez les clés API de vos comptes **Sandbox** (test) et **Live** (production).
        - La clé privée pour le mode **live** commence par **sk_live**.
        - La clé privée pour le mode **sandbox** (test) commence par **sk_sandbox**.

    Copiez ces clés et collez-les dans les champs correspondants dans les paramètres WooCommerce.

    **![(Image montrant où coller les clés API)]**_
  </Step>
  <Step title="Enregistrer les modifications">
    Une fois les clés API correctement entrées, cliquez sur **Enregistrer les modifications** pour finaliser la configuration.

    _**![(Image montrant le bouton "Enregistrer")]**_
  </Step>
  <Step title="Passerelle de paiement prête">
    Félicitations, votre passerelle FedaPay est maintenant configurée et prête à être utilisée pour accepter les paiements de vos clients directement depuis votre boutique WooCommerce !

    _**![(Image montrant la passerelle de paiement active dans WooCommerce)]**_
  </Step>
</Steps>

_**![Vidéo tutoriel : Voir le tutoriel vidéo sur l'intégration de FedaPay avec WooCommerce.]**_

Avec ces étapes simples, vous pouvez maintenant profiter des paiements en ligne rapides et sécurisés grâce à FedaPay sur votre boutique WooCommerce.

## Prestashop

Intégrer FedaPay à votre boutique PrestaShop est simple grâce à notre plugin dédié. Suivez ce guide pour installer et configurer rapidement FedaPay afin de commencer à accepter des paiements en ligne en toute simplicité.

<Steps>
  <Step title="Télécharger et installer le plugin">
    Vous pouvez télécharger le plugin FedaPay pour PrestaShop en cliquant [ici](https://www.fedapay.com/produit/plugin-fedapay-pour-prestashop/?v=ddd70bbaf5be).

    _**![(Image du plugin)]**_

    **Étapes d'installation :**

    1. Accédez à votre tableau de bord PrestaShop.
    2. Cliquez sur **Modules** > **Module Manager**.
    3. Cliquez sur **Installer un module**.

    _**![(Image montrant comment accéder à l'installation des modules)]**_

    4. Téléversez le fichier du plugin que vous avez téléchargé.

    _**![(Image montrant où téléverser le plugin)]**_

    Une fois le plugin téléversé, l'installation démarre automatiquement. Vous recevrez une notification de succès dès que l’installation sera terminée.

    _**[(Image de confirmation d'installation)]**_
  </Step>
  <Step title="Configurer le plugin FedaPay">
    Après l'installation, cliquez sur **Configurer** pour accéder aux paramètres de FedaPay.

    _**![(Image montrant le bouton de configuration)]**_

    Vous pouvez configurer le plugin pour fonctionner en mode test (sandbox) ou en mode live selon vos besoins :

    **Mode Test (Sandbox) :**
      - Connectez-vous à votre compte **FedaPay Sandbox**.
      - Allez dans le menu **API** et copiez la clé secrète de test (sk_sandbox).
      - Collez cette clé dans le champ correspondant sur PrestaShop.

    _**![(Image montrant où coller la clé sandbox)]**_

    **Mode Live :**
    - Connectez-vous à votre compte **FedaPay Live**.
    - Allez dans le menu **API** et copiez la clé secrète live (sk_live).
    - Collez cette clé dans le champ correspondant sur PrestaShop.

    _**![(Image montrant où coller la clé live)]**_

  </Step>
  <Step title="Configurer la devise (FCFA)">
    FedaPay prend en charge uniquement le **FCFA** pour le moment. Vous devez donc configurer cette devise par défaut dans PrestaShop.

    **Étapes pour ajouter le FCFA :**

    1. Allez dans **International** > **Localization**.

    _**![(Image montrant l'accès à la localisation)]**_

    2. Cliquez sur l'onglet **Currencies**.
    3. Cliquez sur **Ajouter une devise**.
    _**![(Image montrant l'ajout d'une devise)]**_
    4. Choisissez **FCFA** dans la liste déroulante.
    5. Définissez le taux de conversion en Euro (si nécessaire).

    _**[(Image montrant le taux de conversion)]**_

    Une fois que vous avez ajouté le **FCFA**, il apparaîtra dans la liste des devises de votre boutique.

    _**![(Image montrant le FCFA dans la liste des devises)]**_

    **Définir le FCFA comme devise par défaut :**

    1. Retournez dans **Localisation**.
    2. Faites défiler la page jusqu'aux paramètres des devises.
    3. Choisissez le FCFA comme devise par défaut.

    _**![(Image montrant comment définir le FCFA par défaut)]**_

  </Step>
  <Step title="Finaliser la configuration">
    Une fois toutes les étapes précédentes terminées, votre boutique PrestaShop est configurée pour accepter les paiements via FedaPay.

    _**![(Image montrant la passerelle FedaPay active dans PrestaShop)]**_
  </Step>
</Steps>

_**![Vidéo tutoriel : Voir le tutoriel vidéo sur l'intégration de FedaPay avec PrestaShop.]**_
